What is DND Shortener?

This project provides a robust URL shortening service that allows users to generate custom or random short URLs, manage expiration dates, track click analytics, and utilize user authentication for secure access. The system integrates MongoDB as the database for efficient data storage and retrieval and uses JWT-based authentication and secure cookie-based sessions for managing user access.
